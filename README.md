# FILE MANAGER - RUST 

Basic 'FILE MANAGER' application to navigate around folder and files. 

## Description

Basic 'FILE MANAGER' application to navigate around folder and files. 
Only for MAC or LINUX os.

### Dependencies

* crossterm

### Executing program

Install dependencies
```
cargo install
```

Run the application
```
cargo run
```
